# Personal Profile Website

Website profil pribadi dengan desain minimalis bertema dark luxury menggunakan HTML dan CSS.

## Deskripsi

Website ini dibuat sebagai tugas praktik pemrograman web. Tampilan mengusung konsep elegan dengan warna gelap dan aksen emas. Halaman menampilkan informasi singkat mengenai profil, tautan referensi, hobi, serta makanan favorit.

## Fitur

- Hero section dengan identitas diri
- Profil singkat
- Tautan menuju Bursa Efek Indonesia (IDX)
- Daftar hobi
- Daftar makanan favorit
- Desain responsif untuk desktop dan mobile
- Smooth scrolling
- Typography menggunakan Google Fonts (Cormorant Garamond & Inter)

## Teknologi

- HTML5
- CSS3
- Google Fonts

## Struktur Website

- **Hero** — Menampilkan nama dan deskripsi singkat.
- **Profil** — Informasi mengenai diri.
- **Tautan** — Link menuju website Bursa Efek Indonesia.
- **Hobi & Favorit** — Daftar hobi dan makanan favorit.
- **Footer** — Informasi pembuat website.

## Tujuan Pembuatan

Website ini dibuat untuk memenuhi tugas praktik mata pelajaran Pemrograman Web sekaligus menjadi latihan dalam membuat tampilan website yang bersih, modern, dan responsif menggunakan HTML dan CSS tanpa framework.
